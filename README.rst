HTTP/2 implementation with hyper-h2_ on Python 3 asyncio.

* Free software: BSD license
* Documentation: https://aioh2.readthedocs.org.

Features
--------

* Asynchronous HTTP/2 client and server
* Managed flow control
* More to come
